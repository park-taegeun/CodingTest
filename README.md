## 💬 커밋 규칙

\`\`\`
type: 문제번호. 문제이름
\`\`\`

| type | 사용 시점 | 예시 |
| :---: | :--- | :--- |
| \`solve\` | 문제 풀이 추가 | \`solve: 1. Two Sum\` |
| \`fix\` | 기존 풀이 수정 | \`fix: 1. Two Sum 예외처리\` |
| \`refactor\` | 풀이 개선 (더 나은 방법) | \`refactor: 1. Two Sum 해시맵 풀이로\` |
| \`docs\` | README 등 문서 수정 | \`docs: 진행 상황 업데이트\` |

> 처음엔 \`solve\`만 써도 충분해. 나머지는 필요할 때 골라 쓰면 돼.
